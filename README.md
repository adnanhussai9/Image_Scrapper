# Image_Scrapper
This project extracts 10 images using the query given in last of scrapper.py file as search_term.

### How to run the project
<br>
Step 1: Dowload the repository as .zip file or clone the repository. <br>
Step 2: Install requirements.txt using command `pip install -r requirements.txt` <br>
Step 3: Run the scraper.py file
